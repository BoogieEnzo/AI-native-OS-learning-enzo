# 2.11–2.16 学习计划

**2.11** 第三章：多道程序与分时多任务  
多道程序放置与加载；任务切换；协作式调度；分时多任务与抢占式调度。完成练习。

虚拟地址与物理地址；页表；SV39 分页机制；内核与用户态地址空间隔离。

**2.13** 第五章：进程  
进程抽象；进程管理；fork/exec/wait；进程调度。

**2.14** 第六章：文件系统  
文件系统布局；inode；目录；文件读写与系统调用。

**2.15** 第七章：进程间通信与 I/O 重定向  
管道；重定向 stdin/stdout；简易 shell 实现。

**2.16** 第八章：并发  
多核；锁；同步原语；并发 bug 与调试。
